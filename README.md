# vue-project-template

## 环境

node v8.11.0+

推荐安装 nvm-windows 

npm install --registry=https://registry.npm.taobao.org

## 插件

* Babel
* Router
* Vuex
* CSS Pre-processors
    ndoe-sass
* Linter/Formatter 
    eslint + standard config
* Unit Testing
    jest
* E2E Testing
    cypress

## 提交规范

cnpm install -g commitizen
commitizen init cz-conventional-changelog --save --save-exact
